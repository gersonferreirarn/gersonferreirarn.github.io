1. Implementação de Algoritmos: Conceitos Fundamentais

Um algoritmo é uma sequência de passos lógicos para resolver um problema. Pense como uma receita de bolo: você precisa seguir as instruções na ordem certa para obter o resultado desejado.

Características de um bom algoritmo:

  • Precisão: Cada passo deve ser claro e sem ambiguidades
  • Finitude: Deve terminar após um número finito de passos
  • Entrada: Recebe dados de entrada para processar
  • Saída: Produz resultados após o processamento
  • Eficácia: Cada passo deve ser simples o suficiente para ser executado

Exemplo de Algoritmo: Calcular a média de dois números

// Passo 1: Receber o primeiro número
ler numero1

// Passo 2: Receber o segundo número
ler numero2

// Passo 3: Calcular a soma
soma = numero1 + numero2

// Passo 4: Calcular a média
media = soma / 2

// Passo 5: Mostrar o resultado
escrever "A média é: " + media

3. Memória, Constantes e Variáveis

A memória do computador é como um armário com muitas gavetas. Cada variável é uma gaveta onde guardamos informações.

Variáveis

Uma variável é um espaço na memória que pode armazenar um valor que pode ser alterado durante a execução do programa.

Constantes

Uma constante é um espaço na memória que armazena um valor que não pode ser alterado após sua definição.

5. Comandos Básicos de Atribuição, Entrada e Saída

Estes são os comandos fundamentais para interagir com dados em um programa.

Comando de Atribuição (=)

Usado para atribuir um valor a uma variável. O valor à direita do "=" é armazenado na variável à esquerda.

int idade;
idade = 25; // Atribuindo 25 à variável idade
String nome = "Carlos"; // Declaração e atribuição

Comando de Entrada (Scanner)

Usado para receber dados do usuário via teclado.

import java.util.Scanner;

Scanner scanner = new Scanner(System.in);

System.out.print("Digite seu nome: ");
String nome = scanner.nextLine();

System.out.print("Digite sua idade: ");
int idade = scanner.nextInt();

Comando de Saída (System.out)

Usado para exibir informações na tela.

System.out.println("Olá, mundo!"); // Exibe e pula linha
System.out.print("Nome: "); // Exibe sem pular linha
System.out.printf("Idade: %d anos\n", idade); // Formatação

Exemplo Completo: Programa de Saudação

Digite seu nome: Maria
Digite sua idade: 20
Olá, Maria!
Você tem 20 anos.
No próximo ano você terá 21 anos.

Módulo 2: Criando seu Primeiro Programa com Saída

1

Entendendo a Estrutura do Código

O NetBeans já criou uma estrutura básica para você. Vamos entender cada parte:

  • package meuprimeiroprograma; - Organiza seu código em uma "pasta"
  • public class Meuprimeiroprograma - Define a classe do programa
  • public static void main(String[] args) - O ponto de entrada do programa
2

Adicionando um Comando de Saída

Dentro do método main, após o comentário, digite o seguinte código:

System.out.println("Olá, mundo! Este é meu primeiro programa em Java.");
Meuprimeiroprograma.java
package meuprimeiroprograma;

public class Meuprimeiroprograma {
  public static void main(String[] args) {
    // TODO code application logic here
    System.out.println("Olá, mundo! Este é meu primeiro programa em Java.");
  }
}

Dica de Digitação

Quando você digitar "System." o NetBeans mostrará uma lista de opções. Use as setas do teclado para selecionar "out" e pressione Enter. Depois digite "." novamente e selecione "println".

3

Salvando o Código

Pressione Ctrl + S ou clique no ícone de disquete na barra de ferramentas para salvar seu código.

4

Executando o Programa

Para rodar seu programa, você tem três opções:

  1. Pressione a tecla F6
  2. Clique no botão verde de "Play" na barra de ferramentas
  3. Vá em Run → Run Project

Módulo 4: Recebendo Dados do Usuário

Criando Projeto com Entrada

2

Importando a Classe Scanner

Para receber dados do usuário, precisamos usar a classe Scanner. Adicione esta linha no início do arquivo, após a linha do package:

import java.util.Scanner;
Entradadados.java
package entradadados;

import java.util.Scanner;

public class Entradadados {
  public static void main(String[] args) {
    // TODO code application logic here
  }
}
3

Criando o Objeto Scanner

Dentro do método main, crie um objeto Scanner para ler dados do teclado:

// Criando objeto Scanner para ler dados
Scanner scanner = new Scanner(System.in);
4

Solicitando e Lendo o Nome

Agora vamos pedir o nome do usuário e ler o que ele digitar:

// Pedindo o nome
System.out.print("Digite seu nome: ");
String nome = scanner.nextLine();

Dica Importante

Use System.out.print() quando quiser que o cursor fique na mesma linha, e System.out.println() quando quiser pular para a próxima linha.

5

Solicitando e Lendo a Idade

Vamos pedir a idade do usuário. Como idade é um número inteiro, usamos nextInt():

// Pedindo a idade
System.out.print("Digite sua idade: ");
int idade = scanner.nextInt();

Cuidado!

Se você usar nextInt() e depois quiser ler texto, precisa adicionar scanner.nextLine() para limpar o buffer. Isso evita problemas na leitura.

6

Exibindo os Dados Lidos

Agora vamos mostrar os dados que o usuário digitou:

// Exibindo os dados
System.out.println("\n--- Dados do Usuário ---");

System.out.println("Idade: " + idade);
System.out.println("No próximo ano você terá " + (idade + 1) + " anos.");

Fechando o Scanner

No final do programa, é uma boa prática fechar o Scanner para liberar recursos:

// Fechando o scanner
scanner.close();

Módulo 5: Criando uma Calculadora Simples

1

Criando o Projeto Calculadora

Crie um novo projeto chamado "CalculadoraSimples". Este projeto vai realizar operações matemáticas básicas.

2

Estrutura Básica

Configure o projeto com a importação do Scanner e a estrutura básica:

import java.util.Scanner;

public class CalculadoraSimples {
  public static void main(String[] args) {
    Scanner scanner = new Scanner(System.in);

    // O código da calculadora virá aqui

    scanner.close();
  }
}
3

Solicitando os Números

Vamos pedir dois números ao usuário:

// Solicitando os números
System.out.print("Digite o primeiro número: ");
double num1 = scanner.nextDouble();

System.out.print("Digite o segundo número: ");
double num2 = scanner.nextDouble();

Por que double?

Usamos double para permitir números decimais. Se usássemos int, não poderíamos calcular com números como 3.14 ou 0.5.

4

Realizando as Operações

Agora vamos realizar as quatro operações básicas:

// Realizando as operações
double soma = num1 + num2;
double subtracao = num1 - num2;
double multiplicacao = num1 * num2;
double divisao = num1 / num2;
double resto = num1 % num2;
5

Exibindo os Resultados

Vamos mostrar todos os resultados de forma organizada:

// Exibindo os resultados
System.out.println("\n--- Resultados ---");
System.out.println(num1 + " + " + num2 + " = " + soma);
System.out.println(num1 + " - " + num2 + " = " + subtracao);
System.out.println(num1 + " * " + num2 + " = " + multiplicacao);
System.out.println(num1 + " / " + num2 + " = " + divisao);
System.out.println("Resto da divisão: " + resto);
6

Tratando Divisão por Zero

Vamos adicionar uma verificação para evitar divisão por zero:

// Verificando divisão por zero
if (num2 != 0) {
  System.out.println(num1 + " / " + num2 + " = " + divisao);
} else {
  System.out.println("Não é possível dividir por zero!");
}

Importante!

Dividir por zero causa um erro no programa. Sempre verifique se o divisor não é zero antes de fazer a divisão.

Módulo 6: Projeto Final - Sistema de Médias

1

Objetivo do Projeto

Vamos criar um programa que calcula a média de um aluno e informa se ele foi aprovado ou reprovado. Este projeto usa tudo que aprendemos!

2

Criando o Projeto

Crie um novo projeto chamado "SistemaMedias". Configure com a importação do Scanner.

3

Solicitando Dados do Aluno

Vamos pedir o nome do aluno e suas três notas:

// Solicitando dados do aluno
System.out.print("Digite o nome do aluno: ");
String nome = scanner.nextLine();

System.out.print("Digite a primeira nota: ");
double nota1 = scanner.nextDouble();

System.out.print("Digite a segunda nota: ");
double nota2 = scanner.nextDouble();

System.out.print("Digite a terceira nota: ");
double nota3 = scanner.nextDouble();
4

Calculando a Média

Calcule a média das três notas:

// Calculando a média
double media = (nota1 + nota2 + nota3) / 3;
5

Determinando a Situação

Use uma estrutura if-else para determinar se o aluno foi aprovado:

// Determinando a situação
String situacao;
if (media >= 7.0) {
  situacao = "APROVADO";
} else if (media >= 5.0) {
  situacao = "RECUPERAÇÃO";
} else {
  situacao = "REPROVADO";
}

Regras do Sistema

Nosso sistema considera:
- Média ≥ 7.0: Aprovado
- 5.0 ≤ Média < 7.0: Recuperação
- Média < 5.0: Reprovado

6

Exibindo o Resultado Final

Mostre todas as informações de forma organizada:

// Exibindo o resultado
System.out.println("\n=== BOLETIM DO ALUNO ===");
System.out.println("Aluno: " + nome);
System.out.println("Nota 1: " + nota1);
System.out.println("Nota 2: " + nota2);
System.out.println("Nota 3: " + nota3);
System.out.printf("Média Final: %.2f\n", media);
System.out.println("Situação: " + situacao);

Formatando Números

Usamos printf com "%.2f" para mostrar a média com apenas 2 casas decimais. Isso deixa o resultado mais bonito!
