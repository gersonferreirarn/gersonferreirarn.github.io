Fundamentos

1. Implementação de Algoritmos: Conceitos Fundamentais

Um algoritmo é uma sequência de passos lógicos para resolver um problema. Pense como uma receita de bolo: você precisa seguir as instruções na ordem certa para obter o resultado desejado.

Características de um bom algoritmo:

  • Precisão: Cada passo deve ser claro e sem ambiguidades
  • Finitude: Deve terminar após um número finito de passos
  • Entrada: Recebe dados de entrada para processar
  • Saída: Produz resultados após o processamento
  • Eficácia: Cada passo deve ser simples o suficiente para ser executado

Exemplo de Algoritmo: Calcular a média de dois números

// Passo 1: Receber o primeiro número
ler numero1

// Passo 2: Receber o segundo número
ler numero2

// Passo 3: Calcular a soma
soma = numero1 + numero2

// Passo 4: Calcular a média
media = soma / 2

// Passo 5: Mostrar o resultado
escrever "A média é: " + media

Usado para atribuir um valor a uma variável. O valor à direita do "=" é armazenado na variável à esquerda.

import java.util.Scanner;

int idade = scanner.nextInt();

System.out.println("Olá, mundo!"); // Exibe e pula linha
System.out.print("Nome: "); // Exibe sem pular linha
System.out.printf("Idade: %d anos\n", idade); // Formatação

Instalando o NetBeans

Passo a passo para instalação:

  1. Baixe o NetBeans: Acesse o site oficial https://netbeans.apache.org/ e faça o download da versão mais recente.
  2. Verifique o Java: Certifique-se de que o Java Development Kit (JDK) está instalado em seu computador. Se não estiver, baixe-o do site da Oracle.
  3. Execute o instalador: Abra o arquivo baixado e siga as instruções do assistente de instalação.
  4. Configure o JDK: Durante a instalação, o NetBeans pode pedir o caminho do JDK. Selecione a pasta onde foi instalado.
  5. Conclua a instalação: Após o término, abra o NetBeans para verificar se tudo funcionou corretamente.

Fazendo a Transição: Do Portugol para Java no NetBeans

A transição do Portugol para o Java no NetBeans pode parecer desafiadora no início, mas seguindo alguns passos, você logo se adaptará:

  1. Entenda a estrutura básica de um programa Java

    Todo programa Java precisa de uma classe e um método main. No Portugol, você começa a escrever algoritmos diretamente, mas em Java é necessário seguir esta estrutura:

    Portugol

    programa
    {
      funcao inicio()
      {
        // Seu código aqui
      }
    }

    Java

    public class MeuPrograma {
      public static void main(String[] args) {
        // Seu código aqui
      }
    }

  2. Aprenda os comandos equivalentes

    Muitos comandos do Portugol têm equivalentes diretos em Java, mas com sintaxe diferente:

    Portugol

    escreva("Olá mundo")
    leia(variavel)
    se (condicao) {
      // código
    }

    Java

    System.out.println("Olá mundo");
    variavel = scanner.nextLine();
    if (condicao) {
      // código
    }

  3. Adapte-se à declaração de variáveis

    Em Java, é necessário declarar o tipo da variável antes de usá-la, e a sintaxe é um pouco diferente:

    Portugol

    inteiro idade = 25
    real altura = 1.75
    caractere letra = 'A'
    logico ativo = verdadeiro

    Java

    int idade = 25;
    double altura = 1.75;
    char letra = 'A';
    boolean ativo = true;

  4. Use o assistente de projetos do NetBeans

    O NetBeans facilita a criação de projetos com estruturas prontas. Use o assistente "New Project" para criar seu primeiro programa Java:

    • Selecione a categoria "Java"
    • Escolha "Java Application"
    • Dê um nome ao seu projeto
    • Marque "Create Main Class" para gerar a estrutura básica

  5. Aproveite os recursos do NetBeans

    O NetBeans oferece recursos que vão além do Portugol Web Studio:

    • Autocompletar inteligente: Pressione Ctrl + Espaço para obter sugestões de código
    • Depuração visual: Use breakpoints para analisar seu código passo a passo
    • Refatoração: Renomeie variáveis e métodos com segurança
    • Geração de código: Crie getters, setters, construtores automaticamente

Primeiro Projeto no NetBeans

Vamos criar um programa simples no NetBeans que você já conhece do Portugol Web Studio: um programa que calcula a média de duas notas.

1

Crie um novo projeto

No NetBeans, vá em File → New Project. Selecione a categoria Java e o tipo de projeto "Java Application". Clique em Next.

2

Configure o projeto

Dê um nome ao projeto, como "CalculaMedia". Marque a opção "Create Main Class" e clique em Finish.

3

Escreva o código

O NetBeans já criou a estrutura básica. Agora, adicione o código dentro do método main:

// Importe a classe Scanner para ler dados do usuário
import java.util.Scanner;

public class CalculaMedia {
  public static void main(String[] args) {
    // Crie um objeto Scanner
    Scanner scanner = new Scanner(System.in);

    // Declare as variáveis
    double nota1, nota2, media;

    // Peça as notas ao usuário
    System.out.print("Digite a primeira nota: ");
    nota1 = scanner.nextDouble();

    System.out.print("Digite a segunda nota: ");
    nota2 = scanner.nextDouble();

    // Calcule a média
    media = (nota1 + nota2) / 2;

    // Mostre o resultado
    System.out.println("A média é: " + media);

    // Feche o scanner
    scanner.close();
  }
}
4

Execute o programa

Pressione F6 ou clique no botão verde de "Run" na barra de ferramentas. O programa será compilado e executado.

import java.util.Scanner;

Solicitando os Números

Vamos pedir dois números ao usuário:

// Solicitando os números

Por que double?

Usamos double para permitir números decimais. Se usássemos int, não poderíamos calcular com números como 3.14 ou 0.5.

4

Realizando as Operações

Agora vamos realizar as quatro operações básicas:

// Realizando as operações
double soma = num1 + num2;
double subtracao = num1 - num2;
double multiplicacao = num1 * num2;
double divisao = num1 / num2;
double resto = num1 % num2;
5

Exibindo os Resultados

Vamos mostrar todos os resultados de forma organizada:

// Exibindo os resultados
System.out.println("\n--- Resultados ---");

System.out.println(num1 + " * " + num2 + " = " + multiplicacao);
System.out.println(num1 + " / " + num2 + " = " + divisao);
System.out.println("Resto da divisão: " + resto);
6

Tratando Divisão por Zero

Vamos adicionar uma verificação para evitar divisão por zero:

// Verificando divisão por zero
if (num2 != 0) {
  System.out.println(num1 + " / " + num2 + " = " + divisao);
} else {
  System.out.println("Não é possível dividir por zero!");
}

Importante!

Dividir por zero causa um erro no programa. Sempre verifique se o divisor não é zero antes de fazer a divisão.

Determinando a Situação

Use uma estrutura if-else para determinar se o aluno foi aprovado:

// Determinando a situação
String situacao;
if (media >= 7.0) {
  situacao = "APROVADO";
} else if (media >= 5.0) {
  situacao = "RECUPERAÇÃO";
} else {
  situacao = "REPROVADO";
}

Regras do Sistema

Nosso sistema considera:
- Média ≥ 7.0: Aprovado
- 5.0 ≤ Média < 7.0: Recuperação
- Média < 5.0: Reprovado

Exibindo o Resultado Final

Mostre todas as informações de forma organizada:

// Exibindo o resultado
System.out.println("\n=== BOLETIM DO ALUNO ===");
System.out.println("Aluno: " + nome);
System.out.println("Nota 1: " + nota1);
System.out.println("Nota 2: " + nota2);
System.out.println("Nota 3: " + nota3);
System.out.printf("Média Final: %.2f\n", media);
System.out.println("Situação: " + situacao);

Formatando Números

Usamos printf com "%.2f" para mostrar a média com apenas 2 casas decimais. Isso deixa o resultado mais bonito!
