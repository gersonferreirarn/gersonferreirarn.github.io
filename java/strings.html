Manipulação de Strings em Java

Principais Métodos e Conceitos

1. Conceitos Básicos de Strings

Em Java, String é uma classe que representa uma sequência de caracteres. É um dos tipos mais utilizados na linguagem e possui diversas funcionalidades para manipulação de texto.

O que é uma String?

Uma String é um objeto que contém uma sequência de caracteres. Em Java, strings são imutáveis, o que significa que uma vez criadas, não podem ser modificadas. Qualquer operação que pareça modificar uma string, na verdade, cria uma nova string.

0
I
1
F
2
R
3
N

Criação de Strings

Existem várias formas de criar strings em Java:

// Forma 1: Literal de String
String nome = "IFRN";

// Forma 2: Usando o construtor
String nome2 = new String("IFRN");

// Forma 3: A partir de um array de caracteres
char[] chars = {'I', 'F', 'R', 'N'};
String nome3 = new String(chars);

String é um Objeto

Em Java, String não é um tipo primitivo como int, double ou char. String é uma classe, e quando criamos uma string, estamos na verdade criando um objeto dessa classe. Isso significa que podemos chamar métodos nesse objeto.

String texto = "Java é uma linguagem orientada a objetos";

// Chamando métodos no objeto String
int tamanho = texto.length(); // Retorna o tamanho da string
String maiuscula = texto.toUpperCase(); // Converte para maiúsculas
boolean contem = texto.contains("linguagem"); // Verifica se contém a substring

Pool de Strings

Java mantém um pool de strings para otimizar o uso de memória. Quando criamos uma string usando um literal, Java verifica se essa string já existe no pool. Se existir, reutiliza a referência em vez de criar uma nova string.

String s1 = "IFRN"; // Criada no pool
String s2 = "IFRN"; Reutiliza a mesma referência do pool
String s3 = new String("IFRN"); // Criada fora do pool

System.out.println(s1 == s2); // true - mesma referência
System.out.println(s1 == s3); // false - referências diferentes
System.out.println(s1.equals(s3)); // true - mesmo conteúdo

Dica Importante

Use sempre o método equals() para comparar o conteúdo de strings, e não o operador ==, que compara referências de objetos.

Concatenação de Strings

Concatenar strings significa unir duas ou mais strings. Em Java, podemos fazer isso de várias formas:

String saudacao = "Olá";
String nome = "Mundo";

// Forma 1: Usando o operador +
String resultado1 = saudacao + ", " + nome + "!";

// Forma 2: Usando o método concat()
String resultado2 = saudacao.concat(", ").concat(nome).concat("!");

System.out.println(resultado1); // "Olá, Mundo!"
System.out.println(resultado2); // "Olá, Mundo!"

Atenção com Concatenação em Loops

Como strings são imutáveis, cada concatenação cria um novo objeto. Para muitas concatenações (especialmente em loops), prefira usar StringBuilder para melhor performance.

2. Métodos de Acesso e Informações

Os métodos de acesso e informações permitem obter dados sobre uma string, como seu tamanho, caracteres específicos e verificar se está vazia.

length()

O método length() retorna o número de caracteres em uma string. É importante notar que este é um método, não uma propriedade como em arrays (que usam .length sem parênteses).

String texto = "Java";
int tamanho = texto.length();
System.out.println("Tamanho: " + tamanho); // Imprime "Tamanho: 4"

charAt(int index)

O método charAt(int index) retorna o caractere na posição especificada pelo índice. Os índices em Java começam em 0, então o primeiro caractere está no índice 0.

String texto = "IFRN";
char primeiro = texto.charAt(0); // 'I'
char terceiro = texto.charAt(2); // 'R'

System.out.println("Primeiro caractere: " + primeiro);
System.out.println("Terceiro caractere: " + terceiro);

Cuidado com os Índices!

Se você tentar acessar um índice que não existe (maior ou igual ao tamanho da string ou negativo), será lançada uma exceção StringIndexOutOfBoundsException.

isEmpty()

O método isEmpty() retorna true se a string estiver vazia (tamanho 0) e false caso contrário. Uma string vazia é diferente de uma string nula.

String vazia = "";
String naoVazia = "IFRN";
String nula = null;

System.out.println("String vazia está vazia? " + vazia.isEmpty()); // true
System.out.println("String não vazia está vazia? " + naoVazia.isEmpty()); // false
// System.out.println("String nula está vazia? " + nula.isEmpty()); // Erro: NullPointerException

3. Métodos de Substrings e Modificações

Os métodos de substring e modificações permitem extrair partes de uma string ou criar versões modificadas da string original. Lembre-se que, devido à imutabilidade das strings, esses métodos sempre retornam uma nova string.

substring(int início)

O método substring(int início) retorna uma nova string que é uma substring da string original, começando no índice especificado até o final da string.

String texto = "Programação em Java";
String sub = texto.substring(13);
System.out.println(sub); // "Java"

substring(int início, int fim)

O método substring(int início, int fim) retorna uma nova string que é uma substring da string original, começando no índice especificado por início e terminando no índice fim - 1.

String texto = "Programação em Java";
String sub = texto.substring(0, 11);
System.out.println(sub); // "Programação"

Importante

No método substring(início, fim), o caractere no índice fim não é incluído na substring. Por exemplo, "Java".substring(0, 2) retorna "Ja", não "Jav".

replace(old, new)

O método replace(old, new) substitui todas as ocorrências da sequência de caracteres old pela sequência new na string.

String texto = "Java é uma linguagem. Java é popular.";
String novoTexto = texto.replace("Java", "Python");
System.out.println(novoTexto); // "Python é uma linguagem. Python é popular."

toUpperCase()

O método toUpperCase() converte todos os caracteres da string para maiúsculas.

String texto = "Programação em Java";
String maiusculas = texto.toUpperCase();
System.out.println(maiusculas); // "PROGRAMAÇÃO EM JAVA"

toLowerCase()

O método toLowerCase() converte todos os caracteres da string para minúsculas.

String texto = "Programação em Java";
String minusculas = texto.toLowerCase();
System.out.println(minusculas); // "programação em java"

trim()

O método trim() remove os espaços em branco do início e do final da string.

String texto = " Java ";
String textoTrim = texto.trim();
System.out.println("Tamanho original: " + texto.length()); // 9
System.out.println("Tamanho após trim(): " + textoTrim.length()); // 4
System.out.println("Texto após trim(): '" + textoTrim + "'"); // 'Java'

4. Métodos de Busca e Comparação

Os métodos de busca e comparação permitem encontrar substrings dentro de uma string, comparar strings e verificar se uma string começa ou termina com determinada sequência de caracteres.

equals()

O método equals() compara o conteúdo de duas strings, retornando true se forem iguais e false caso contrário. Esta comparação é sensível a maiúsculas e minúsculas.

String s1 = "Java";
String s2 = "Java";
String s3 = "java";

System.out.println(s1.equals(s2)); // true
System.out.println(s1.equals(s3)); // false (diferença de maiúsculas/minúsculas)

equalsIgnoreCase()

O método equalsIgnoreCase() compara o conteúdo de duas strings, ignorando diferenças entre maiúsculas e minúsculas.

String s1 = "Java";
String s2 = "JAVA";
String s3 = "java";

System.out.println(s1.equalsIgnoreCase(s2)); // true
System.out.println(s1.equalsIgnoreCase(s3)); // true

startsWith()

O método startsWith() verifica se a string começa com a sequência de caracteres especificada.

String texto = "Programação em Java";
System.out.println(texto.startsWith("Pro")); // true
System.out.println(texto.startsWith("pro")); // false (sensível a maiúsculas/minúsculas)
System.out.println(texto.startsWith("Java")); // false

endsWith()

O método endsWith() verifica se a string termina com a sequência de caracteres especificada.

String texto = "Programação em Java";
System.out.println(texto.endsWith("Java")); // true
System.out.println(texto.endsWith("java")); // false (sensível a maiúsculas/minúsculas)
System.out.println(texto.endsWith("Pro")); // false

indexOf()

O método indexOf() retorna o índice da primeira ocorrência da sequência de caracteres especificada. Se a sequência não for encontrada, retorna -1.

String texto = "Java é uma linguagem. Java é popular.";
System.out.println(texto.indexOf("Java")); // 0
System.out.println(texto.indexOf("linguagem")); // 11
System.out.println(texto.indexOf("Python")); // -1 (não encontrado)

lastIndexOf()

O método lastIndexOf() retorna o índice da última ocorrência da sequência de caracteres especificada. Se a sequência não for encontrada, retorna -1.

String texto = "Java é uma linguagem. Java é popular.";
System.out.println(texto.lastIndexOf("Java")); // 23
System.out.println(texto.lastIndexOf("linguagem")); // 11
System.out.println(texto.lastIndexOf("Python")); // -1 (não encontrado)

Os métodos de divisão e conversão permitem dividir uma string em partes menores ou converter strings para outros tipos de dados e vice-versa.

split(String regex)

O método split(String regex) divide a string em um array de substrings com base em uma expressão regular (regex) que especifica o delimitador.

String texto = "Java,Python,C++,JavaScript";
String[] linguagens = texto.split(",");

for (String linguagem : linguagens) {
  System.out.println(linguagem.trim());
}

// Saída:
// Java
// Python
// C++
// JavaScript

O método toCharArray() converte a string em um array de caracteres.

for (int i = 0; i < caracteres.length; i++) {

6. Imutabilidade e StringBuilder

Um dos conceitos mais importantes sobre strings em Java é a sua imutabilidade. Vamos entender o que isso significa e como trabalhar com strings de forma eficiente quando precisamos fazer muitas modificações.

O que é Imutabilidade?

Imutabilidade significa que, uma vez criado, um objeto não pode ser modificado. No caso de strings em Java, qualquer operação que parece modificar uma string, na verdade, cria uma nova string com as modificações desejadas.

1

Criamos uma string:

String s = "Java";
2

Quando tentamos modificar a string:

s = s + " é divertido";
3

Na verdade, uma nova string é criada:

// A string original "Java" permanece inalterada // Uma nova string "Java é divertido" é criada // A variável s agora referencia a nova string

Problemas de Performance com Concatenação

Devido à imutabilidade, cada operação de concatenação cria um novo objeto string. Isso pode causar problemas de performance quando fazemos muitas concatenações, especialmente em loops.

Forma Ineficiente

String resultado = "";
for (int i = 0; i < 10000; i++) {
  resultado += i + ",";
}
// Cria 10000 objetos String!

Forma Eficiente

StringBuilder sb = new StringBuilder();
for (int i = 0; i < 10000; i++) {
  sb.append(i).append(",");
}
String resultado = sb.toString();
// Cria apenas 1 objeto String!

StringBuilder

Para resolver os problemas de performance com strings imutáveis, Java oferece a classe StringBuilder. Ao contrário de String, StringBuilder é mutável e permite modificar seu conteúdo sem criar novos objetos.

Principais Métodos do StringBuilder

  • append(): Adiciona conteúdo ao final do StringBuilder
  • insert(): Insere conteúdo em uma posição específica
  • delete(): Remove caracteres entre duas posições
  • replace(): Substitui caracteres entre duas posições
  • reverse(): Inverte o conteúdo do StringBuilder
  • toString(): Converte o StringBuilder para String

Exemplos Práticos com StringBuilder

sb.append(" é uma ");

StringBuffer vs StringBuilder

Java também oferece a classe StringBuffer, que é semelhante ao StringBuilder, mas com uma diferença importante: StringBuffer é thread-safe (seguro para uso em ambientes concorrentes), enquanto StringBuilder não é.

td>Rápida
Característica String StringBuilder StringBuffer
Imutabilidade Imutável Mutável Mutável
Performance Lenta para muitas modificações Rápida, mas um pouco mais lenta que StringBuilder
Thread-safety Thread-safe (imutabilidade garante isso) Não thread-safe Thread-safe
Quando usar Para strings que não serão modificadas Para strings que serão modificadas em um único thread Para strings que serão modificadas em múltiplos threads

Dica de Performance

Use StringBuilder sempre que precisar fazer muitas modificações em strings, especialmente em loops. Para operações simples com poucas concatenações, usar o operador + é aceitável e mais legível.

    relatorio += "\n";

    double[] vendas = {1200.50, 850.75, 1500.00, 950.25};
    double total = 0;

    for (int i = 0; i < vendas.length; i++) {
      relatorio += "Venda " + (i + 1) + ": R$ " + vendas[i] + "\n";
      total += vendas[i];
    }

    relatorio += "\nTotal: R$ " + total;
    System.out.println(relatorio);
  }
