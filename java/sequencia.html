Estruturas Sequenciais em Java

Vetores e Matrizes

1. Conceitos Gerais de Estruturas Sequenciais

As estruturas sequenciais (ou arrays) são coleções de dados do mesmo tipo, armazenados de forma contígua na memória. Elas permitem armazenar múltiplos valores em uma única variável, acessados através de índices.

O que são estruturas sequenciais?

Em Java, arrays são objetos que contêm um número fixo de valores de um único tipo. O comprimento de um array é estabelecido quando o array é criado e não pode ser alterado posteriormente.

Tipagem Uniforme

Um dos princípios fundamentais dos arrays em Java é a tipagem uniforme: todos os elementos em um array devem ser do mesmo tipo de dado. Isso garante consistência e eficiência no armazenamento e manipulação dos dados.

// Exemplo de arrays com diferentes tipos
int[] numeros = new int[5]; // Array de inteiros
String[] nomes = new String[3]; // Array de strings
double[] valores = new double[10]; // Array de decimais
boolean[] flags = new boolean[2]; // Array de booleanos

Endereçamento por Índice

Os elementos em um array são acessados através de índices, que são números inteiros que representam a posição do elemento no array. Em Java, os índices começam em 0 e vão até (tamanho - 1).

Vantagens das Estruturas Sequenciais

  • Acesso rápido: O acesso a qualquer elemento é feito em tempo constante O(1) através do índice.
  • Memória contígua: Os elementos são armazenados em posições consecutivas de memória, o que otimiza o cache.
  • Simplicidade: São fáceis de entender e implementar.
  • Previsibilidade: O tamanho é fixo e conhecido antecipadamente.

Desvantagens das Estruturas Sequenciais

  • Tamanho fixo: Não é possível alterar o tamanho de um array após sua criação.
  • Inserção e remoção: São operações custosas, pois exigem deslocamento de elementos.
  • Alocação contígua: Pode ser difícil encontrar grandes blocos contíguos de memória.

2. Vetores (Arrays Unidimensionais)

Vetores (ou arrays unidimensionais) são estruturas sequenciais que organizam os elementos em uma única dimensão, como uma lista. Cada elemento é acessado através de um único índice.

Declaração de Vetores

Em Java, existem duas formas principais de declarar um vetor:

// Forma 1: Declaração e criação separadas
int[] numeros; // Declaração
numeros = new int[5]; // Criação com tamanho 5

// Forma 2: Declaração e criação juntas
String[] nomes = new String[3]; // Array de strings com tamanho 3

// Forma 3: Inicialização com valores
double[] valores = {3.5, 7.2, 4.8, 9.1}; // Array com valores predefinidos

Inicialização de Vetores

Após a declaração, os vetores precisam ser inicializados. Se não forem inicializados explicitamente, receberão valores padrão conforme o tipo:

Tipo Valor Padrão Exemplo
int, byte, short, long 0 int[] v = new int[3]; // [0, 0, 0]
double, float 0.0 double[] v = new double[2]; // [0.0, 0.0]
boolean false boolean[] v = new boolean[4]; // [false, false, false, false]
char '\u0000' char[] v = new char[2]; // ['\u0000', '\u0000']
Objetos (String, etc.) null String[] v = new String[3]; // [null, null, null]

Acesso por Índice

Os elementos de um vetor são acessados através do índice, que começa em 0 e vai até (tamanho - 1):

String[] nomes = {"Ana", "Carlos", "Mariana", "Pedro"};

// Acesso aos elementos
System.out.println(nomes[0]); // Imprime "Ana"
System.out.println(nomes[2]); // Imprime "Mariana"

// Modificação de elementos
nomes[1] = "Carolina"; // Altera o segundo elemento
System.out.println(nomes[1]); // Imprime "Carolina"

Percurso com FOR e FOR-EACH

Existem duas formas principais de percorrer todos os elementos de um vetor:

FOR Tradicional

int[] numeros = {5, 10, 15, 20, 25};

for (int i = 0; i < numeros.length; i++) {
  System.out.println("Elemento " + i + ": " + numeros[i]);
}

FOR-EACH

int[] numeros = {5, 10, 15, 20, 25};

for (int numero : numeros) {
  System.out.println("Elemento: " + numero);
}

Operações Comuns com Vetores

Soma e Média

double[] notas = {7.5, 8.0, 6.5, 9.0, 7.0};
double soma = 0;

for (double nota : notas) {
  soma += nota;
}

double media = soma / notas.length;
System.out.println("Soma: " + soma);
System.out.println("Média: " + media);

Maior e Menor Elemento

int[] valores = {42, 17, 93, 8, 56};
int maior = valores[0]; // Assume o primeiro como maior

for (int i = 1; i < valores.length; i++) {

Contagem por Condição

int[] numeros = {2, 7, 4, 9, 6, 1, 8};
int pares = 0;
int impares = 0;

for (int num : numeros) {
  if (num % 2 == 0) {
    pares++;
  } else {
    impares++;
  }
}

System.out.println("Quantidade de pares: " + pares);
System.out.println("Quantidade de ímpares: " + impares);

Busca Linear

String[] nomes = {"Ana", "Carlos", "Mariana", "Pedro"};
String busca = "Mariana";
boolean encontrado = false;
int posicao = -1;

for (int i = 0; i < nomes.length; i++) {

  System.out.println("Nome encontrado na posição: " + posicao);
} else {
  System.out.println("Nome não encontrado");
}

Erros Comuns com Vetores

ArrayIndexOutOfBoundsException

Este é o erro mais comum ao trabalhar com arrays em Java. Ocorre quando tentamos acessar um índice que não existe no array:

int[] numeros = {10, 20, 30}; // Array com 3 elementos (índices 0, 1, 2)

// Acessos válidos
System.out.println(numeros[0]); // 10
System.out.println(numeros[2]); // 30

// Acesso inválido - gera ArrayIndexOutOfBoundsException
System.out.println(numeros[3]); // Erro! O índice máximo é 2

Cuidado com os Índices!

Lembre-se sempre que os índices em Java começam em 0 e vão até (tamanho - 1). Um array com 5 elementos tem índices de 0 a 4.

3. Matrizes (Arrays Bidimensionais)

Matrizes (ou arrays bidimensionais) são estruturas sequenciais que organizam os elementos em duas dimensões, como uma tabela ou grade. Cada elemento é acessado através de dois índices: um para a linha e outro para a coluna.

Declaração de Matrizes

Em Java, as matrizes são declaradas de forma semelhante aos vetores, mas com dois pares de colchetes:

// Forma 1: Declaração e criação separadas
int[][] matriz; // Declaração
matriz = new int[3][4]; // Criação: 3 linhas e 4 colunas

// Forma 2: Declaração e criação juntas
double[][] notas = new double[5][3]; // 5 linhas e 3 colunas

// Forma 3: Inicialização com valores
String[][] nomes = {
  {"Ana", "Carlos"},
  {"Mariana", "Pedro"},
  {"João", "Maria"}
};

Inicialização de Matrizes

Assim como os vetores, as matrizes podem ser inicializadas de várias formas:

// Inicialização com valores padrão
int[][] matriz = new int[3][4]; // Todos os elementos serão 0

// Inicialização com valores explícitos
int[][] numeros = {
  {1, 2, 3},
  {4, 5, 6},
  {7, 8, 9}
};

// Inicialização linha por linha
int[][] matriz2 = new int[3][];
matriz2[0] = new int[2]; // Primeira linha com 2 colunas
matriz2[1] = new int[3]; // Segunda linha com 3 colunas
matriz2[2] = new int[2]; // Terceira linha com 2 colunas

Acesso com Dois Índices

Os elementos de uma matriz são acessados através de dois índices: o primeiro para a linha e o segundo para a coluna:

int[][] matriz = {
  {1, 2, 3},
  {4, 5, 6},
  {7, 8, 9}
};

// Acesso aos elementos
System.out.println(matriz[0][0]); // Imprime 1 (primeira linha, primeira coluna)
System.out.println(matriz[1][2]); // Imprime 6 (segunda linha, terceira coluna)
System.out.println(matriz[2][1]); // Imprime 8 (terceira linha, segunda coluna)

// Modificação de elementos
matriz[0][1] = 10; // Altera o elemento da primeira linha, segunda coluna
System.out.println(matriz[0][1]); // Imprime 10

Percurso com Loops Aninhados

Para percorrer todos os elementos de uma matriz, precisamos de loops aninhados: um para as linhas e outro para as colunas:

int[][] matriz = {
  {1, 2, 3},
  {4, 5, 6},
  {7, 8, 9}
};

// Percurso com loops for tradicionais
for (int i = 0; i < matriz.length; i++) { // Para cada linha
  for (int j = 0; j < matriz[i].length; j++) { // Para cada coluna
    System.out.print(matriz[i][j] + " ");
  }
  System.out.println(); // Nova linha após cada linha da matriz
}

Operações Comuns com Matrizes

Soma Total

int[][] matriz = {
  {1, 2, 3},
  {4, 5, 6},
  {7, 8, 9}
};
int soma = 0;

for (int i = 0; i < matriz.length; i++) {
  for (int j = 0; j < matriz[i].length; j++) {
    soma += matriz[i][j];
  }
}

System.out.println("Soma total: " + soma);

Soma por Linha e Coluna

int[][] matriz = {
  {1, 2, 3},
  {4, 5, 6},
  {7, 8, 9}
};

// Soma por linha
for (int i = 0; i < matriz.length; i++) {
  int somaLinha = 0;
  for (int j = 0; j < matriz[i].length; j++) {
    somaLinha += matriz[i][j];
  }
  System.out.println("Soma da linha " + i + ": " + somaLinha);
}

// Soma por coluna
for (int j = 0; j < matriz[0].length; j++) {
  int somaColuna = 0;
  for (int i = 0; i < matriz.length; i++) {
    somaColuna += matriz[i][j];
  }
  System.out.println("Soma da coluna " + j + ": " + somaColuna);
}

Diagonais

int[][] matriz = {
  {1, 2, 3},
  {4, 5, 6},
  {7, 8, 9}
};
int somaPrincipal = 0;
int somaSecundaria = 0;

// Diagonal principal (i == j)
for (int i = 0; i < matriz.length; i++) {
  somaPrincipal += matriz[i][i];
}

// Diagonal secundária (i + j == n - 1)
for (int i = 0; i < matriz.length; i++) {
  somaSecundaria += matriz[i][matriz.length - 1 - i];
}

System.out.println("Soma da diagonal principal: " + somaPrincipal);
System.out.println("Soma da diagonal secundária: " + somaSecundaria);

Busca de Elementos

int[][] matriz = {
  {1, 2, 3},
  {4, 5, 6},
  {7, 8, 9}
};
int valorBusca = 5;
boolean encontrado = false;
int linha = -1;
int coluna = -1;

for (int i = 0; i < matriz.length; i++) {
  for (int j = 0; j < matriz[i].length; j++) {
    if (matriz[i][j] == valorBusca) {
      encontrado = true;
      linha = i;
      coluna = j;
      break; // Sai do loop interno
    }
  }
  if (encontrado) {
    break; // Sai do loop externo
  }
}

if (encontrado) {
  System.out.println("Valor " + valorBusca + " encontrado na posição [" + linha + "][" + coluna + "]");
} else {
  System.out.println("Valor " + valorBusca + " não encontrado na matriz");
}

Erros Comuns com Matrizes

Confusão entre Linha e Coluna

Um erro comum é inverter a ordem dos índices, acessando matriz[coluna][linha] em vez de matriz[linha][coluna]:

int[][] matriz = {
  {1, 2, 3},
  {4, 5, 6},
  {7, 8, 9}
};

// Acesso correto
System.out.println(matriz[1][2]); // Imprime 6 (linha 1, coluna 2)

// Acesso incorreto (pode gerar erro ou valor inesperado)
System.out.println(matriz[2][1]); // Imprime 8 (linha 2, coluna 1)

Tamanhos Irregulares (Arrays Aninhados)

Em Java, é possível criar matrizes onde cada linha tem um número diferente de colunas. Isso pode causar erros se não for tratado corretamente:

// Matriz irregular
int[][] matrizIrregular = new int[3][];
matrizIrregular[0] = new int[2]; // 2 colunas
matrizIrregular[1] = new int[3]; // 3 colunas
matrizIrregular[2] = new int[1]; // 1 coluna

// Preenchendo a matriz
matrizIrregular[0][0] = 1;
matrizIrregular[0][1] = 2;
matrizIrregular[1][0] = 3;
matrizIrregular[1][1] = 4;
matrizIrregular[1][2] = 5;
matrizIrregular[2][0] = 6;

// Tentativa de acesso inválido (gera ArrayIndexOutOfBoundsException)
matrizIrregular[2][1] = 7; // Erro! A terceira linha tem apenas 1 coluna

Cuidado com os Índices!

Lembre-se sempre que o primeiro índice representa a linha e o segundo a coluna. Além disso, em matrizes irregulares, cada linha pode ter um número diferente de colunas.

4. Comparação entre Arrays Unidimensionais e Bidimensionais

Arrays unidimensionais (vetores) e bidimensionais (matrizes) têm características distintas que os tornam adequados para diferentes tipos de problemas. Vamos comparar essas estruturas em vários aspectos.

Quantidade de Índices

A diferença mais óbvia entre vetores e matrizes é a quantidade de índices necessários para acessar seus elementos:

Vetor (1D)

int[] vetor = new int[5];

// Acesso com um índice
vetor[0] = 10;
int valor = vetor[3];

Matriz (2D)

int[][] matriz = new int[3][4];

// Acesso com dois índices
matriz[0][1] = 20;
int valor = matriz[2][3];

Complexidade do Percurso

A complexidade para percorrer todos os elementos também difere significativamente:

Vetor (1D)

int[] vetor = {1, 2, 3, 4, 5};

// Percurso com um loop
for (int i = 0; i < vetor.length; i++) {
  System.out.println(vetor[i]);
}

Matriz (2D)

int[][] matriz = {
  {1, 2, 3},
  {4, 5, 6}
};

// Percurso com loops aninhados
for (int i = 0; i < matriz.length; i++) {
  for (int j = 0; j < matriz[i].length; j++) {
    System.out.print(matriz[i][j] + " ");
  }
  System.out.println();
}

Uso Típico: Listas vs Tabelas

A escolha entre vetores e matrizes geralmente depende da natureza dos dados que estamos representando:

1D
Listas
Ideal para representar listas de elementos, como notas de alunos, nomes, temperaturas, etc.
2D
Tabelas
Ideal para representar tabelas, como planilhas, matrizes matemáticas, imagens, etc.

Exemplos de Aplicação

Vetores (1D)

  • Lista de notas: Armazenar as notas de um aluno em diferentes disciplinas.
  • Temperaturas diárias: Registrar a temperatura máxima de cada dia do mês.
  • Nomes de produtos: Manter uma lista de produtos em estoque.
  • Resultados de pesquisa: Armazenar as respostas de um questionário.

Matrizes (2D)

  • Notas de turma: Armazenar as notas de vários alunos em várias disciplinas.
  • Imagens digitais: Representar pixels de uma imagem (linha x coluna x cor).
  • Matrizes matemáticas: Realizar operações de álgebra linear.
  • Mapas de assentos: Organizar os assentos de um cinema ou teatro.

Comparação Detalhada

Transformação entre 1D e 2D

É possível converter entre representações 1D e 2D. Por exemplo, uma matriz pode ser "achatada" em um vetor:

// Matriz para vetor (achatamento)
int[][] matriz = {
  {1, 2, 3},
  {4, 5, 6}
};

int[] vetor = new int[matriz.length * matriz[0].length];
int index = 0;

for (int i = 0; i < matriz.length; i++) {
  for (int j = 0; j < matriz[i].length; j++) {
    vetor[index++] = matriz[i][j];
  }
}

// Resultado: [1, 2, 3, 4, 5, 6]

E também é possível fazer o inverso, transformando um vetor em uma matriz:

// Vetor para matriz
int[] vetor = {1, 2, 3, 4, 5, 6};
int linhas = 2;
int colunas = 3;

int[][] matriz = new int[linhas][colunas];

for (int i = 0; i < linhas; i++) {
  for (int j = 0; j < colunas; j++) {
    matriz[i][j] = vetor[i * colunas + j];
  }
}

// Resultado: [[1, 2, 3], [4, 5, 6]]

5. Boas Práticas com Estruturas Sequenciais

Trabalhar com arrays em Java requer atenção a alguns detalhes para garantir código limpo, eficiente e livre de erros. Vamos explorar as melhores práticas ao utilizar estruturas sequenciais.

Verificar Tamanho com .length

A propriedade .length é fundamental para trabalhar com arrays em Java. Ela retorna o tamanho do array e deve ser usada sempre que precisarmos percorrer ou verificar os limites do array:

int[] numeros = {10, 20, 30, 40, 50};

// Boa prática: usar .length para determinar o limite
for (int i = 0; i < numeros.length; i++) {
  System.out.println(numeros[i]);
}

// Evitar: usar valores fixos (hardcoded)
for (int i = 0; i < 5; i++) { // Ruim se o tamanho mudar
  System.out.println(numeros[i]);
}

Dica Importante

Use sempre .length em vez de valores fixos. Isso torna seu código mais flexível e menos propenso a erros quando o tamanho do array mudar.

Usar Nomes Claros para Índices

Embora seja comum usar i, j, k como nomes para índices, em loops complexos ou com propósito específico, é melhor usar nomes mais descritivos:

// Menos claro
for (int i = 0; i < matriz.length; i++) {
  for (int j = 0; j < matriz[i].length; j++) {
    matriz[i][j] = 0;
  }
}

// Mais claro
for (int linha = 0; linha < matriz.length; linha++) {
  for (int coluna = 0; coluna < matriz[linha].length; coluna++) {
    matriz[linha][coluna] = 0;
  }
}

Inicialização Explícita

Sempre inicialize seus arrays explicitamente, mesmo que Java atribua valores padrão. Isso torna o código mais legível e evita surpresas:

// Boa prática: inicialização explícita
int[] contadores = new int[10];
Arrays.fill(contadores, 0); // Preenche todos com 0

// Para matrizes
int[][] matriz = new int[5][5];
for (int[] linha : matriz) {
  Arrays.fill(linha, 0);
}

Evitar Valores Mágicos (Usar Constantes)

Valores mágicos são números literais no código sem explicação. É melhor usar constantes com nomes significativos:

// Ruim: valores mágicos
int[][] tabuleiro = new int[8][8];
if (posicaoX >= 0 && posicaoX < 8 && posicaoY >= 0 && posicaoY < 8) {
  // ...
}

// Bom: usando constantes
public static final int TAMANHO_TABULEIRO = 8;

int[][] tabuleiro = new int[TAMANHO_TABULEIRO][TAMANHO_TABULEIRO];
if (posicaoX >= 0 && posicaoX < TAMANHO_TABULEIRO &&
    posicaoY >= 0 && posicaoY < TAMANHO_TABULEIRO) {
  // ...
}

Validar Índices Antes do Acesso

Para evitar ArrayIndexOutOfBoundsException, sempre valide os índices antes de acessar os elementos, especialmente quando os índices vêm de entrada do usuário ou cálculos:

int[] vetor = {10, 20, 30};
int indice = 5; // Índice inválido

// Boa prática: validar antes de acessar
if (indice >= 0 && indice < vetor.length) {
  System.out.println(vetor[indice]);
} else {
  System.out.println("Índice inválido!");
}

Usar Enhanced For-Loop Quando Possível

Quando você só precisa percorrer os elementos sem precisar do índice, use o enhanced for-loop (for-each) para código mais limpo:

int[] numeros = {1, 2, 3, 4, 5};
int soma = 0;

// Mais limpo com for-each
for (int numero : numeros) {
  soma += numero;
}

// Menos limpo com for tradicional
for (int i = 0; i < numeros.length; i++) {
  soma += numeros[i];
}

Documentar Arrays Complexos

Para arrays multidimensionais ou com significado específico, documente a estrutura e o propósito de cada dimensão:

/**
* Matriz de notas onde:
* - Primeira dimensão: índice do aluno (0 a 29)
* - Segunda dimensão: índice da disciplina (0 a 4)
*/
double[][] notasAlunos = new double[30][5];

Considerar o Uso de Collections para Tamanhos Dinâmicos

Se você precisa de uma estrutura que possa crescer dinamicamente, considere usar Collections como ArrayList em vez de arrays:

// Com array (tamanho fixo)
int[] numeros = new int[10]; // Tamanho fixo

// Com ArrayList (tamanho dinâmico)
ArrayList numeros = new ArrayList<>();
numeros.add(10); // Adiciona elementos dinamicamente
numeros.add(20);
numeros.add(30);

Atenção

Arrays são mais eficientes em termos de memória e performance para tamanhos fixos, mas Collections são mais flexíveis para tamanhos variáveis.

Atividade Prática

Reescreva o código abaixo aplicando as boas práticas que aprendemos:

public class ExemploRuim {
  public static void main(String[] args) {
    int[] v = new int[10];
    for (int i = 0; i < 10; i++) {
      v[i] = i * 2;
    }
    int soma = 0;
    for (int i = 0; i < 10; i++) {
      soma += v[i];
    }
    System.out.println("Soma: " + soma);
  }
}
