Estruturas de Repetição em Java

For, While e Do While

1. Introdução às Estruturas de Repetição

As estruturas de repetição (também conhecidas como laços ou loops) são recursos fundamentais em programação que permitem executar um bloco de código repetidamente enquanto uma determinada condição for verdadeira.

Por que usar estruturas de repetição?

Imagine que você precisa imprimir os números de 1 a 100. Sem estruturas de repetição, você teria que escrever 100 linhas de código! Com um laço, você pode fazer isso em apenas algumas linhas.

Componentes de uma Estrutura de Repetição

Todas as estruturas de repetição possuem componentes essenciais:

  • Inicialização: Onde definimos o valor inicial da variável de controle
  • Condição: Uma expressão lógica que determina quando o loop deve continuar ou parar
  • Atualização: Modificação do valor da variável de controle a cada iteração
  • Corpo do Loop: O bloco de código que será repetido

Exemplo Comparativo

Vamos ver como cada estrutura de repetição pode ser usada para imprimir os números de 1 a 5:

Com FOR

for (int i = 1; i <= 5; i++) {
  System.out.println(i);
}

Com WHILE

int i = 1;
while (i <= 5) {
  System.out.println(i);
  i++;
}

Com DO-WHILE

int i = 1;
do {
  System.out.println(i);
  i++;
} while (i <= 5);

2. Estrutura de Repetição FOR

A estrutura for é uma das mais utilizadas em programação. Ela é ideal quando sabemos exatamente quantas vezes o loop deve ser executado.

Sintaxe do FOR

for (inicialização; condição; atualização) {
  // bloco de código a ser repetido
}

Componentes do FOR

  • Inicialização: Executada apenas uma vez, no início do loop. Geralmente usada para declarar e inicializar a variável de controle.
  • Condição: Verificada antes de cada iteração. Se for verdadeira, o loop continua; se for falsa, o loop termina.
  • Atualização: Executada após cada iteração. Geralmente usada para modificar a variável de controle.

Fluxo de Execução do FOR

Passo a Passo do Funcionamento

1
A inicialização é executada (apenas uma vez)
2
A condição é avaliada
3
Se a condição for verdadeira, o bloco de código é executado
4
A atualização é executada
5
Volta para o passo 2 (avaliação da condição)

Exemplo 1: Contagem Simples

Um exemplo básico que imprime os números de 1 a 10:

for (int i = 1; i <= 10; i++) {
  System.out.println(i);
}

Exemplo 2: Contagem Regressiva

Um exemplo que faz uma contagem regressiva de 10 a 1:

for (int i = 10; i >= 1; i--) {
  System.out.println(i);
}
System.out.println("Feliz Ano Novo!");

Exemplo 3: Percorrendo Arrays

O for é muito utilizado para percorrer elementos de um array:

String[] nomes = {"Ana", "Carlos", "Mariana", "Pedro"};

for (int i = 0; i < nomes.length; i++) {
  System.out.println("Nome " + (i + 1) + ": " + nomes[i]);
}

Exemplo 4: For Aprimorado (For-Each)

Java oferece uma versão simplificada do for para percorrer coleções e arrays:

String[] nomes = {"Ana", "Carlos", "Mariana", "Pedro"};

for (String nome : nomes) {
  System.out.println("Olá, " + nome + "!");
}

Exemplo 5: Loops Aninhados

Podemos usar um for dentro de outro for. Isso é muito útil para trabalhar com matrizes ou para criar padrões:

// Imprimindo uma tabuada
for (int i = 1; i <= 10; i++) {
  System.out.println("Tabuada do " + i + ":");
  for (int j = 1; j <= 10; j++) {
    System.out.println(i + " x " + j + " = " + (i * j));
  }
  System.out.println(); // linha em branco
}

Dica de Boa Prática

Use nomes significativos para as variáveis de controle do loop. Em vez de usar sempre "i", "j", "k", tente usar nomes que indiquem o propósito da variável, como "linha", "coluna", "indice", etc.

Cuidado com Loops Infinitos!

Se a condição nunca se tornar falsa, o loop continuará executando indefinidamente. Por exemplo:

// Loop infinito - não faça isso!
for (int i = 1; i > 0; i++) {
  System.out.println("Isso nunca vai parar!");
}

3. Estrutura de Repetição WHILE

A estrutura while (enquanto) executa um bloco de código repetidamente enquanto uma determinada condição for verdadeira. Diferente do for, o while não tem uma contagem explícita de iterações.

Sintaxe do WHILE

while (condição) {
  // bloco de código a ser repetido
  // (geralmente inclui a atualização da variável de controle)
}

Fluxo de Execução do WHILE

Passo a Passo do Funcionamento

1
A condição é avaliada
2
Se a condição for verdadeira, o bloco de código é executado
3
Volta para o passo 1 (avaliação da condição)
4
Se a condição for falsa, o loop termina

Exemplo 1: Contagem Simples

Um exemplo que imprime os números de 1 a 10 usando while:

int contador = 1;

while (contador <= 10) {
  System.out.println(contador);
  contador++; // incrementa o contador
}

Exemplo 2: Validação de Entrada

O while é muito útil para validar entradas do usuário:

import java.util.Scanner;

Scanner scanner = new Scanner(System.in);
int idade;

do {
  System.out.print("Digite sua idade (entre 1 e 120): ");
  idade = scanner.nextInt();
  if (idade < 1 || idade > 120) {
    System.out.println("Idade inválida! Tente novamente.");
  }
} while (idade < 1 || idade > 120);

System.out.println("Idade registrada: " + idade);

Exemplo 3: Soma de Valores

Um programa que soma valores digitados pelo usuário até que ele digite 0:

import java.util.Scanner;

Scanner scanner = new Scanner(System.in);
double soma = 0;
double valor;

System.out.println("Digite valores para somar (0 para sair):");

while (true) { // loop infinito
  System.out.print("Valor: ");
  valor = scanner.nextDouble();

  if (valor == 0) {
    break; // sai do loop
  }

  soma += valor; // mesmo que soma = soma + valor
}

System.out.println("A soma dos valores é: " + soma);

Exemplo 4: Jogo de Adivinhação

Um jogo simples onde o usuário tenta adivinhar um número:

import java.util.Scanner;
import java.util.Random;

Scanner scanner = new Scanner(System.in);
Random random = new Random();

int numeroSecreto = random.nextInt(100) + 1; // número entre 1 e 100
int tentativa;
int contagem = 0;

System.out.println("Tente adivinhar o número entre 1 e 100:");

do {
  System.out.print("Digite seu palpite: ");
  tentativa = scanner.nextInt();
  contagem++;

  if (tentativa < numeroSecreto) {
    System.out.println("Muito baixo! Tente novamente.");
  } else if (tentativa > numeroSecreto) {
    System.out.println("Muito alto! Tente novamente.");
  }
} while (tentativa != numeroSecreto);

System.out.println("Parabéns! Você acertou em " + contagem + " tentativas!");

Cuidado com Loops Infinitos!

Como a condição é verificada apenas no início, se a condição nunca se tornar falsa, o loop continuará para sempre. Por exemplo:

// Loop infinito - não faça isso!
int x = 5;
while (x > 0) {
  System.out.println("Isso nunca vai parar!");
  // esquecemos de decrementar x
}

Dica de Boa Prática

Em loops while, certifique-se de que a condição eventualmente se tornará falsa. Isso geralmente significa que você precisa modificar alguma variável dentro do loop que afeta a condição.

4. Estrutura de Repetição DO-WHILE

A estrutura do-while (faça-enquanto) é semelhante ao while, mas com uma diferença crucial: o bloco de código é executado pelo menos uma vez, antes mesmo de verificar a condição.

Sintaxe do DO-WHILE

do {
  // bloco de código a ser repetido
  // (geralmente inclui a atualização da variável de controle)
} while (condição);

Fluxo de Execução do DO-WHILE

Passo a Passo do Funcionamento

1
O bloco de código é executado (pelo menos uma vez)
2
A condição é avaliada
3
Se a condição for verdadeira, volta para o passo 1
4
Se a condição for falsa, o loop termina

Diferença entre WHILE e DO-WHILE

A principal diferença é que o do-while executa o bloco de código pelo menos uma vez, enquanto o while pode não executar nenhuma vez se a condição for falsa inicialmente.

WHILE

int x = 10;
while (x < 5) {
  System.out.println("Isso não será impresso");
  x++;
}
// Nada é impresso, pois a condição é falsa desde o início

DO-WHILE

int x = 10;
do {
  System.out.println("Isso será impresso uma vez");
  x++;
} while (x < 5);
// A mensagem é impressa uma vez, mesmo com a condição falsa

Exemplo 1: Menu de Opções

O do-while é perfeito para menus, onde queremos mostrar as opções pelo menos uma vez:

import java.util.Scanner;

Scanner scanner = new Scanner(System.in);
int opcao;

do {
  System.out.println("\n=== MENU PRINCIPAL ===");
  System.out.println("1. Cadastrar usuário");
  System.out.println("2. Listar usuários");
  System.out.println("3. Buscar usuário");
  System.out.println("4. Sair");
  System.out.print("Escolha uma opção: ");
  opcao = scanner.nextInt();

  switch (opcao) {
    case 1:
      System.out.println("Opção de cadastro selecionada");
      break;
    case 2:
      System.out.println("Opção de listagem selecionada");
      break;
    case 3:
      System.out.println("Opção de busca selecionada");
      break;
    case 4:
      System.out.println("Saindo do sistema...");
      break;
    default:
      System.out.println("Opção inválida!");
  }
} while (opcao != 4);

Exemplo 2: Validação de Entrada

O do-while também é útil para validar entradas, garantindo que o usuário digite um valor válido:

import java.util.Scanner;

Scanner scanner = new Scanner(System.in);

do {
  System.out.print("Digite sua idade (entre 18 e 65): ");
  idade = scanner.nextInt();

  }
} while (idade < 18 || idade > 65);

System.out.println("Idade válida: " + idade);

Dica de Boa Prática

Use o do-when quando você precisa que um bloco de código seja executado pelo menos uma vez, independentemente da condição. É especialmente útil para menus e validação de entrada.

Cuidado com o Ponto e Vírgula!

No do-while, é fácil esquecer o ponto e vírgula no final da condição. Isso causará um erro de compilação:

// Errado - falta o ponto e vírgula
do {
  // código
} while (condição) // Erro de compilação aqui

// Correto
do {
  // código
} while (condição); // Ponto e vírgula necessário

5. Comandos BREAK e CONTINUE

Os comandos break e continue são usados para controlar o fluxo de execução dentro de estruturas de repetição, permitindo interromper ou pular iterações.

O Comando BREAK

O comando break é usado para sair imediatamente de uma estrutura de repetição (for, while ou do-while), transferindo o controle para a primeira linha após o loop.

Exemplo 1: BREAK em um FOR

for (int i = 1; i <= 10; i++) {
  System.out.println("Número: " + i);

  if (i == 5) {
    System.out.println("Interrompendo o loop...");
    break; // sai do loop
  }
}
System.out.println("Loop encerrado");

Exemplo 2: BREAK em um WHILE

import java.util.Scanner;

Scanner scanner = new Scanner(System.in);
int soma = 0;

while (true) { // loop infinito
  System.out.print("Digite um número (0 para sair): ");
  int numero = scanner.nextInt();

  if (numero == 0) {
    break; // sai do loop quando o usuário digita 0
  }

  soma += numero;
}

System.out.println("A soma dos números é: " + soma);

O Comando CONTINUE

O comando continue é usado para pular a iteração atual de um loop e continuar com a próxima iteração. Quando o continue é executado, o código restante da iteração atual é ignorado.

Exemplo 3: CONTINUE em um FOR

// Imprimindo apenas números ímpares de 1 a 10
for (int i = 1; i <= 10; i++) {
  if (i % 2 == 0) { // se o número for par
    continue; // pula para a próxima iteração
  }
  System.out.println("Número ímpar: " + i);
}

Exemplo 4: CONTINUE em um WHILE

import java.util.Scanner;

Scanner scanner = new Scanner(System.in);
int contadorPositivos = 0;
int contadorNegativos = 0;

for (int i = 1; i <= 10; i++) {
  System.out.print("Digite o número " + i + ": ");
  int numero = scanner.nextInt();

  if (numero == 0) {
    continue; // pula zeros
  }

  if (numero > 0) {
    contadorPositivos++;
  } else {
    contadorNegativos++;
  }
}

System.out.println("Quantidade de números positivos: " + contadorPositivos);
System.out.println("Quantidade de números negativos: " + contadorNegativos);

BREAK com Rótulos (Labels)

Java permite o uso de rótulos com o comando break para sair de loops aninhados. O rótulo é um identificador seguido de dois pontos (:).

Exemplo 5: BREAK com Rótulos

// Procurando um valor em uma matriz
int[][] matriz = {
  {1, 2, 3},
  {4, 5, 6},
  {7, 8, 9}
};

int valorProcurado = 5;
boolean encontrado = false;

// Rótulo para o loop externo
loopExterno: for (int i = 0; i < matriz.length; i++) {
  for (int j = 0; j < matriz[i].length; j++) {
    if (matriz[i][j] == valorProcurado) {
      encontrado = true;
      System.out.println("Valor " + valorProcurado + " encontrado na posição [" + i + "][" + j + "]");
      break loopExterno; // sai de ambos os loops
    }
  }
}

if (!encontrado) {
  System.out.println("Valor " + valorProcurado + " não encontrado na matriz");
}

CONTINUE com Rótulos

Assim como o break, o continue também pode ser usado com rótulos para pular iterações de loops externos.

Exemplo 6: CONTINUE com Rótulos

// Imprimindo números de 1 a 30, exceto múltiplos de 2 e 3
loopExterno: for (int i = 1; i <= 30; i++) {
  for (int j = 2; j <= 3; j++) {
    if (i % j == 0) {
      continue loopExterno; // pula para a próxima iteração do loop externo
    }
  }
  System.out.print(i + " ");
}
System.out.println();

Dica de Boa Prática

Embora break e continue sejam úteis, o uso excessivo pode tornar o código difícil de entender. Tente estruturar seu código para minimizar a necessidade desses comandos.

Cuidado com BREAK e CONTINUE!

O uso incorreto de break e continue pode levar a comportamentos inesperados. Certifique-se de entender exatamente como eles afetam o fluxo do seu programa.
